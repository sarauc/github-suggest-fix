PR Review AI Assistant — Architecture

Chrome Extension · FastAPI :8765 · BM25 RAG · claude-sonnet-4-5 · SSE Streaming

Chrome Extension (MV3)

  • • content.js — button injection, panel UI, SSE client
  • • background.js — backend health ping every 30s
  • • popup.html/js — API keys, re-index trigger
  • • Only activates for PR authors on github.com/*/pull/*
  • • MutationObserver (300ms debounce) for GitHub SPA
  • • abortCurrentStream() on panel close / navigation

FastAPI Backend :8765

  • • Binds to 127.0.0.1 only — no network exposure
  • • CORS restricted to github.com + localhost
  • • BM25 (rank_bm25) — no embedding model required
  • • Context: diff hunk → file → RAG → imports → summary
  • • Prompt: mentor style, 2-4 approaches, no paste-ready code
  • • Rotating logs (10 MB, 2 backups) via RotatingFileHandler

External Integrations

  • • GitHub REST API v3 — PAT Bearer auth (repo scope)
  • • Anthropic Claude API — claude-sonnet-4-5, SSE stream
  • • Credentials passed per-request — never stored server-side
  • • Errors surfaced as SSE error events with machine codes
  • • Outdated comment detection via commit_id vs head SHA
  • • 5-min indexing timeout → graceful fallback (no-RAG)
